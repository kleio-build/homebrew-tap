# Homebrew Tap for Kleio

This is the official Homebrew tap for [Kleio CLI](https://github.com/kleio-build/kleio-cli).

## Installation

```bash
# Install directly (recommended)
brew install kleio-build/tap/kleio

# Or tap first, then install
brew tap kleio-build/tap
brew install kleio
```

## Update

```bash
brew update
brew upgrade kleio
```

## Uninstall

```bash
brew uninstall kleio
brew untap kleio-build/tap  # optional
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `kleio` | Kleio CLI — capture work discovered during development; includes `kleio mcp` for MCP |

## About Kleio

Kleio CLI captures work items and backlog signals during development. Learn more at [github.com/kleio-build/kleio-cli](https://github.com/kleio-build/kleio-cli).
